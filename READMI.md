#Selenium Based Framework

##Infrastructure
* TestBase
* Config
* Test Logger
* Data
* Utils
* Webdriver manager

##Installing

Clone the repo:
```
git clone git@github.com:ValerieDy/selenium-based-framework.git
```
Use java version 1.8.0_181

Add JUnit configuration


## Running the tests

#### Run all tests
```
mvn test
```
